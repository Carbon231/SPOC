## 12.5

### 学生注册

- 请求方式：`POST`

- URL：`StudentRegister/`

- 请求参数：

  ```json
  {
    "student_id": "String",
    "student_name": "String",
    "password": "String",
    "password_confirm": "String"
  }
  ```

* 返回值：

  * 注册成功：

    ```json
    {
      "code": 201,
      "message": "注册成功！"
    }
    ```
  
  * 注册失败：
  
    ```json
    {
      "code": 400,
      "error": "密码不正确！"
    }
    ```
    
    或者
    
    ```json
    {
      "code": 400,
      "error": "用户名已存在！"
    }
    ```



### 学生登录

- 请求方式：`POST`
- URL：`StudentLogin/`

- 请求参数：

  ```json
  {
    "student_id": "String",
    "password": "String"
  }
  ```

* 返回值：

  * 登录成功：

    ```json
    {
      "code": 200,
      "message": "登录成功！"
    }
    ```

  * 登录失败：

    ```json
    {
      "code": 401,
      "error": "用户名不存在或密码不正确！"
    }
    ```
